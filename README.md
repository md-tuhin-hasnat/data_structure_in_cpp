# Custom Data Structures in C++

Hi there!
This is a personal project where I'm building some basic data structures in C++ from scratch — like `Vector`, `Set`, and more (coming soon!).
The goal is to learn how these containers work internally, and not rely fully on the STL.

---

## ✅ What's Inside

| Data Structure | Description                                                                            |
| -------------- | -------------------------------------------------------------------------------------- |
| `Vector`     | A resizable array, like `std::vector`. It supports push, pop, indexing, and sorting. |
| `Set`        | A simple set that stores**unique** values. Built using a dynamic array.          |
